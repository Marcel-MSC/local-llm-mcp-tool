===========================================
COMMAND TO INSTALL llama-cpp-python
==========================================

COPY AND PASTE THIS COMMAND IN YOUR TERMINAL:

pip install llama-cpp-python --extra-index-url https://abetlen.github.io/llama-cpp-python/whl/cpu

===========================================

IMPORTANT:
- Do NOT run just "pip install llama-cpp-python"
- You MUST include the --extra-index-url
- This downloads a pre-compiled version (doesn't try to compile)

===========================================

If the command above doesn't work, try:

pip install llama-cpp-python==0.2.20 --extra-index-url https://abetlen.github.io/llama-cpp-python/whl/cpu

===========================================

Or use the automated scripts:
- PowerShell: .\install_llama.ps1
- CMD: install_llama.bat

===========================================
